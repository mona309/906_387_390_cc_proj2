How to Use:

  • Draw on the canvas - your strokes will be replicated across all replicas
  • Open multiple browser tabs to see real-time collaboration
  • Click "Simulate Failover" to see instructions for testing leader failure
  • The status panel shows current RAFT state, leader, and log consistency

Simulate Failover Instructions

To simulate leader failure and observe automatic failover:

  1. Open a terminal in the project directory
  2. Run: docker ps to see running containers
  3. Find the current leader container (shown in status panel)
  4. Run: docker stop <leader-container-name>
  5. Watch the status panel update as a new leader is elected
  6. Drawing will continue seamlessly with the new leader
